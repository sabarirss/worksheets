How to Use This Page

  1. Click any prompt box to automatically copy it to clipboard
  2. Go to Bing Image Creator (free)
  3. Paste the prompt and click "Create"
  4. Download the image and rename it to the filename shown (e.g., animal_01.png)
  5. Save to the images/stories/ folder
  6. Repeat for all 100 stories!

Tip: Start with just a few to test, then continue with the rest.
